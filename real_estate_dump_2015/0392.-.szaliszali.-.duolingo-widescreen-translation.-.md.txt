duolingo-widescreen-translation
===============================

An userscript for Duolingo Immersion to utilize more screen real estate

Features
--------

* Increase document width
* Automatically open DuoBot translations
* Add a button to copy DuoBot translation to the input textarea

Installation
------------

* Install [GreaseMonkey](http://www.greasespot.net/) / [TamperMonkey](http://tampermonkey.net/) extension
* Install the [script](duolingo-widescreen-translation.user.js)
