# Property4U-iOS-Sample-App
Property4U iOS Sample App with ASP.NET Web API 2.2, OAuth and OWIN

## Introduction:
Property4U iOS Real Estate Sample App completely equipped with Buying, Selling, Renting and Bidding features.

## System Requirements:
  - Mountain Lion OS X 10.8.5
  - Xcode 5.0.2

## System Specifications:
  - iOS 6
  - Property4U
  - Authentication with ASP.NET WEB API 2.2
  - OAuth and OWIN
  - ASP.NET Identity 2.0
  - EntityFramework 6
