== Description ==

Geodigs is a way for real estate agents to connect with clients and help them discover their dream home.

* Real time MLS data from local Colorado sources
* Share important documents securely through the built-in Document Store
* Help your clients stay up-to-date with listing alerts
* Have access to a variety of search tools to help your clients narrow down their search
* Manage featured listings easily
* Compatable with all currently existing dsIDX Themes
* And much more!


== Installation ==

**REQUIRES A GEODIGS ACCOUNT**

1. Upload `geodigs` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Geodigs Agent ID and Geodigs Agent Code
