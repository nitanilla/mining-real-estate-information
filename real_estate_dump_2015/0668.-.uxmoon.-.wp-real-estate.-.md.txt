## WordPress Theme for real estate website

WordPress theme for a small real estate business using the starter theme from underscores.me called _s and the Foundation Framework for front-end development.

Website: [CIP Consultora Constructora](http://consultoracip.com)

## Tools
  * [Fontello](http://fontello.com/): Use icon fonts instead of fontawesome.io to reduce file size.
  * [Grunt](http://gruntjs.com/): concat, uglify, spritesmith, sass and watch.
  * [Foundation](http://foundation.zurb.com/): Front-end development Framework with Mobile First approach.

## Featured
  * Custom post type to manage properties.
  * Each property have amenities, description, photo gallery and Google Maps.
  * Search properties using Google Maps.
  * Generic form for each property with automatic response.
  * Added Google Analytics.

## WordPress Tweaks
  * Replace WordPress bundled jQuery with [jQuery CDN](https://code.jquery.com/).
  * Removed additional code from head section.
  * Removed WP emoji code from markup.

## Foundation Js
  * Foundation Off Canvas for mobile navigation.
  * Foundation Equalizer to force columns with the same height.
