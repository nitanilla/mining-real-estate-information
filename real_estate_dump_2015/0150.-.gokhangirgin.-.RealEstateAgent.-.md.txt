A simple Real Estate Agent with Google Maps
===============

Geographic Information Systems Course Project

The map bounded for Izmir/Turkey. You can change the bounds from Views/RealEstate/Index.cshtml
And you need to put google maps script with your credintals to Views/Shared/Layout.cshtml

Technologies

ASP.NET MVC 3
Entity Framework 4.1
SQL Server Express 2008
jQuery 1.7.2
Google Maps API

The project can be viewed at http://stdhomes.ieu.edu.tr/gokhangirgin/se340.html if it is still working. :)
