# RealEstateWebApp
My Web platform to manage a Real Estate Agency
A .NET 4.6 c#/asp.net/jquery platform for your real estate.

This web app provides a useful platform to manage a Real Estate.
- Front end web app:
    - Responsive Design with Bootstrap 3, jQuery and RESTful queries to the Web Service.
    - Customizable design: home page, logos, colors
- Back end web app:
    - Web Service made with .NET 4.6 and json response.
