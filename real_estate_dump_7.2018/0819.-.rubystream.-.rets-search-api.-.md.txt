# her-realtor real estate API

RETS stands for Real Estate Transaction Standard which is a set of documents describing 
how software developers can program their software according to standards agreed upon 
by the Real Estate Standards Organization (RESO).

## Get started

Install dependences
```
$ npm i
```

Start the server
```
npm start
```
