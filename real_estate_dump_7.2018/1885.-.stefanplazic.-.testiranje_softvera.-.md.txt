The purpose of this project is to make a real life system for estate trading. 
The site has a couple of roles of users (like: Advertiser,Customers,Administrators and Moderators.) The customers are able to look up for 
estates and to buy them, if they prefere it.
The advertisers have the purpose of selling estates to potencional buyers
The admins and moderators are the ones who run this site , and take care about any user conflicts.

#technology
Site is done in Spring framework. It user Spring data to do ORM to MYSQL database. All comunications are done via rest services. 
All users are given the token verification  , during login.
