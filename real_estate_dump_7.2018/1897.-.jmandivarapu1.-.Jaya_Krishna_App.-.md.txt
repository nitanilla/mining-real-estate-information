## Installation Instructions

1. Clone this repository:
    ```
    git clone https://github.com/jmandivarapu1/Jaya_Krishna_App
    ```

1. Navigate to the `Jaya_Krishna_App` directory:
    ```
    cd Jaya_Krishna_App
    ```

1. Install the npm dependencies:
    ```
    npm install
    ```

1. Install the native dependencies:
    ```
    react-native link
    ```

## Run in the iOS Emulator

1. Type the following command :

    ```
    react-native run-ios
    ```

## Testing

To Run tests type :
 ```
    npm run test
 ```
