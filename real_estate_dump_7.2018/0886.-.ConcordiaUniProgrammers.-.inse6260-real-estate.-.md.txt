# inse6260-real-estate
a real estate web application for INSE 6260 course - Group project 
This project is moved from:

https://code.google.com/p/inse6260-real-estate/

Three concordia master students have contribution in this project. this is a java web application as a DDD and Onion Architecture that uses hibernate, DI Spring, jersey, junit in back-end and angularjs, jquery, Bootstrap, kendo ui in front-end.
