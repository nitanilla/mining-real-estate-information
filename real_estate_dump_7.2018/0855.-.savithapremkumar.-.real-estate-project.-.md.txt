# real-estate-project
Real estate project repository is for maintenance of real estate projects. This enables the project maintainers to add new buildings, the blocks in the buildings
update the kind of blocks, the number of bedrooms of apartment and so on.

#Build
Project is built using mvn clean install

#Technology
1. Spring-boot
2. MySQL DB
3. Test uses H2 in memory DB.

#Deployment
Run spring boot jar.
