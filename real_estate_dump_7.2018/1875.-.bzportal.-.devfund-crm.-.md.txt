# DevFund CRM for Real Estate Developers

> In Business we often have lots of contacts and contracts.
> It is impossible to check every details on various windows and places.
> This app is making all visible in one place and provide lots of benefit.

## Overview

- Manage potential real estate deals
- Multi processing multiple deals via email

## Technical Base

#### Server

- Node.js
- Express.js
- JWT Authorization
- MongoDB

#### Client

- Node.js
- Webpack
- Angular4
- TypeScript
- Rxjs
- Material/Bootstrap

#### Run

- npm install
- npm install -g @angular/cli
- ng build
- npm start
- server: localhost:3000/api
- client: localhost:3000
