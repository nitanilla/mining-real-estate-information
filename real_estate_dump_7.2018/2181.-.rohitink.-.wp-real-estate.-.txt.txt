WP Real Estate
=========

WP Real Estate is a Very Simple, Stylish and Fast Loading WordPress theme Designed for Real estate firms and agents. Although Designed keeping properties in mind, this theme is so flexible it can be used by Lawyers, Doctors, Artists and More! And yes, this theme is 100% Responsive and will work on all your devices from Mobiles, Tablets to Desktops. 

Demo Here: http://demo.rohitink.com/wpre/

Getting Started
---------------

How to Install the Theme?

1. Go to WP Admin > Appearance > Themes > Add New.
2. Search for "WP Real Estate".
3. Click on Install and Activate.

Alternatively,

1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the wpre.zip file, in which you found this README.txt
3. Click on Install and Activate.

Setting it Up
-------------

This theme is built using Customizer. So to Configure theme settings like Featured Content Area, Sliders, Social Icons, etc from 
	
	Dashboard > Appearance > Customizer
