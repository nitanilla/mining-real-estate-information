Coast Realty Denman and Hornby Island - Admin Panel
=====================

Created a content management system for use along side real estate listings for all the real estate on Denman and Hornby Islands. CMS was created using PHP and MySQL for server side and HTML / CSS for front end application.

</ > Tech: PHP, MySQL, HTML, CSS, Adobe Photoshop
