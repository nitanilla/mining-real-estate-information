moodle-editor_butinymce
=======================

Customized version of TinyMCE for Moodle@BU

Special, cleaner version of TinyMCE for Moodle- saves a lot of screen real estate, especially on question editing screens.

Authored by Kyle Temkin, working for Binghamton University <http://www.binghamton.edu>

To install Moodle 2.1+ using git, execute the following commands in the root of your Moodle install:

    git clone git://github.com/ktemkin/moodle-editor_butinymce.git lib/editor/butinymce
    echo '/lib/editor/butinymce' >> .git/info/exclude

Or, extract the following zip in your_moodle_root/lib/editor/:

    https://github.com/ktemkin/moodle-editor_butinymce/zipball/master
