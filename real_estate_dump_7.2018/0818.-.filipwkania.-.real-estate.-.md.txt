# Real Estate CMS
Student project for Real Estate CMS

Featured requirements:
- Single Page Application ( AJAX )
- Front-end technologies: HTML, CSS, SASS, JavaScript and jQuery
- Back-end technologies: PHP, MongoDB 
- CRUD on real estates and users
- Google maps
- Multiple images upload
- Desktop notifications
- Sounds
- Validation front-end, back-end ( own coded validation library )
- Sessions
- Mailing

JS files placed in pages/js/
APIs placed in services/
