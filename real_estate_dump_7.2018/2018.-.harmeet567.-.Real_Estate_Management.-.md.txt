# Real_Estate_Management
Real Estate Management application developed in ASP.NET , C# and SQL SERVER 2012 
Instructions to run this project:
1) Install visual studio 2012 and above .
2) Open this project in it.
3) Create a new database in it named RealEstate and then run the scripts provided in the Database-updateScripts.txt file.
4) Change the connection string in the web config file.
