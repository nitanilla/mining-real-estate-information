# SmartZillow

Real Estate Search and Value Prediction System
* Designed and implemented the system using Service-oriented Architecture (SOA) with JSON-RPC
* Developed a web service with caching using Node.js/Express, Redis, Bing Map API and Material Design
* Built a web scraping system to collect real-time property information with MongoDB and RabbitMQ
* Implemented a real estate value online prediction system using TensorFlow
