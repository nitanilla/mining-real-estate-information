# weekend5-real-estate
This application allows the user to access and view real estate rentals and properties for sale. New sales and rentals can be added to the database and to the DOM. 

## Built With

-Angular
-Mongo
-Mongoose
-Bootstrap
-Javascript
-CSS
-Node
-Express
