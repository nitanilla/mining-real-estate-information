#Real Estate Spring Project

##DB connections
Data connections are made via environmental variables. The variables that are required for RealEstate are:

```
   	HOST
	MSSQL_REALESTATE_HST
	USERNAME
	MSSQL_REALESTATE_UN
	PASSWORD
	MSSQL_REALESTATE_PW 
```

##File paths needed for uploading Real Estate Files

Configuration file is env.properties. DEV and TEST environment uses different one.

``` 
	$SERVER_PATH/property
	$SERVER_PATH/project

```
