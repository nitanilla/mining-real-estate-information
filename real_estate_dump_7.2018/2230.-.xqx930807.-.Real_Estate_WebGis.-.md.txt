# Real Estate Web Geographical Information system



UITFONCIER is a web geographic information system and it's designed to store, manage, analyze and finally estimate the value of a real estate property using :

    HTML/CSS
    Bootstrap
    JS / jQuery
    Openlayers 3
    PHP 5
    PostgreSQL/PostGIS
    Geoserver
    AJAX


UITFONCIER is shared under the GNU Public Licence 
(GPLv2, v3, and the AGPLv3). This means that the code can 
be freely copied and distributed, and costs nothing to use 
in other GPL applications.
