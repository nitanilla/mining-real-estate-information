== Description ==

Lava real estate manager plugin is for preperty

This plugin enables you to use the power and flexibility of WordPress to generate optimized pages for your real estate listings.

Intro page : http://lava-code.com/real-estate/

Documentation : http://lava-code.com/real-estate/documentation/

Demo site : http://lava-code.com/real-estate-demo/


== Installation ==

Install via Search:

1. In WordPress admin, visit Plugins > Add New
2. Search for "lava real estate manager"
3. Click the "Install Now" link.
4. Once installed, click the "Activate Plugin" link.
5. Visit settings to setup pages

Install via Upload:

1. In WordPress admin, visit "Plugins" > "Add New" > "Upload"
2. Upload lava-real-estate-manager.zip file
3. Once uploaded, click the "Activate Plugin" link
4. Visit settings to setup pages


That's it!
If you need more documentation, please visit here ( http://lava-code.com/real-estate/documentation/ )


== Frequently Asked Questions ==

= Can I use my existing WordPress theme? =

Yes! Lava real estate manager works out-of-the-box with nearly every WordPress theme.

= Will this work on WordPress multisite? =

Yes! If your WordPress installation has multisite enabled, Lava real estate manager will work on it.
