# estate-reel
This is a web app that creates a social network for users to share info about their real estate properties. 
The objective is that people can go, post their properties and display them. Other uses can go, watch and contact.
This web app has messenger, user profiles, admin functions, feeds, in-deep search of the DB by filters and few other features.
It is done in PHP, MySQL, JS, JQuery, HTML and CSS.
