# Planet Page
A concept page for an off world real estate company.

## Objectives
1. Make a visually appealing site.
2. Show use of CSS by using custom fonts and buttons.
3. Add css animations for on hover events.
4. Make mobile version of site.
5. Ensure cross-browser reliability.
6. Use a free domain.
7. Host using Heroku.
