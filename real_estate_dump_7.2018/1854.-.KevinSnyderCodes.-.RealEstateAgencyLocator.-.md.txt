# Real Estate Agency Locator

A simple React app that locates real estate agencies near multiple addresses using Google Maps API.

## Features

- Search for closest agencies to one or multiple addresses.
- Results are not limited to one municipality. Search across the entire United States!
  - Results work best when addresses are closer together.

## Usage

Simply navigate to the root directory of the project and run:

```shell
npm start
```

Then visit `localhost:3000` in a web browser.

A live version of the app can also be found at https://polar-reef-66567.herokuapp.com
